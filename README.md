## Run

- git clone

```shell
$ git clone https://github.com/tail-passengers/tail-passengers.git
$ cd tail-passengers
$ git submodule init
$ git submodule update
$ make
```

- update

```shell
# in the tail-passengers directory
$ git submodule update --remote
$ make
```
